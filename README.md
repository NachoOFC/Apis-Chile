# 🇨🇱 Listado de APIs Públicas en Chile

Recopilación de APIs públicas chilenas para distintos tipos de servicios digitales. La idea es tener en un solo lugar las APIs que sirven para hacer proyectos, prototipos o simplemente cachar qué datos están disponibles en Chile.

Última actualización: **JuLio 2026**

SI QUIERES COLABORAR REVISA EL OTRO ARCHIVO MD Contribuir.md

## 🏛️ Servicios Públicos / Gobierno

- [API Biblioteca del Congreso](http://www.leychile.cl/Consulta/legislacion_abierta_web_service): Leyes, Proyectos de Ley y Normas.
- [API Mercado Público](http://api.mercadopublico.cl/): Acceso en tiempo real a licitaciones, órdenes de compra, compradores y proveedores del Estado.
- [API Portal ChileAtiende](https://www.chileatiende.gob.cl/desarrolladores): API del Portal de Servicios del Estado.
- [Plataforma Ley de Lobby](https://www.leylobby.gob.cl/docs.html): Integra los contenidos del portal de Ley de Lobby del Gobierno de Chile.
- [API Comisión Nacional de Energía](http://api.cne.cl/): Información generada dentro de la CNE. Usado en [Bencina en línea](http://www.bencinaenlinea.cl/).
- [Compras Transparentes](http://www.comprastransparentes.cl/api): Explora las transacciones entre el Estado y empresas vía plataforma de compras públicas.
- [ChileCompra](https://www.chilecompra.cl/api/): API de Mercado Público para licitaciones, órdenes de compra y proveedores del Estado.
- [Seguimiento de pedidos de Correos de Chile](https://github.com/josemontesp/correos-chile-npm): Módulo npm para seguimiento de pedidos de Correos de Chile.
- [FeriadosApp](https://www.feriadosapp.com/api/): API sin restricciones con todos los feriados legales de Chile.
- [Feriados Chile — API](https://feriados-cl.netlify.app/): Feriados oficiales Chile 2026 y 2027.
- [Turnos de Farmacia](http://datos.gob.cl/dataset/farmacias-en-chile): Listado JSON de farmacias y turnos nocturnos legales desde FARMANET del Ministerio de Salud.
- [API REST del Congreso](http://congresorest.appspot.com/): Servicio open source que mapea el contenido del Congreso para desarrolladores.
- [SuperSalud Desarrolladores](https://apis-portal.superdesalud.gob.cl/): API de la Superintendencia de Salud con datos de salud, isapres y prestadores. Requiere registro.
- [DataInnovación CORFO](https://datainnovacion.cl/api): API REST con datos públicos de más de 10.300 proyectos beneficiados por InnovaChile.
- [Codificador Digital — Gobierno Digital](https://codificador.digital.gob.cl/como-usar-la-api/): API de la Secretaría de Gobierno Digital.

## 💰 Economía

- [Indicadores económicos diarios — mindicador.cl](http://mindicador.cl/): UF, dólar, euro, IPC y más en formato JSON. Open source.
- [Indicadores económicos Chile — findic.cl](https://findic.cl/): API gratuita con indicadores diarios e históricos, además de archivos CSV.
- [Indicadores del Día](http://indicadoresdeldia.cl/pages/code/): Datos del Banco Central actualizados cada hora en JSON, XML, CSV y JS.
- [API CMF](https://api.cmfchile.cl): Indicadores financieros y reportes bancarios de la Comisión para el Mercado Financiero.
- [API BDE — Banco Central](https://si3.bcentral.cl/estadisticas/Principal1/web_services/index.htm): Bases de datos estadísticos del Banco Central de Chile. Con ejemplos en Python, R y C#.
- [DolarApi.com](https://dolarapi.com/docs/chile/): Precio del dólar y otras monedas en Chile.
- [Bolsa de Santiago](https://startup.bolsadesantiago.com/): APIs para desarrolladores de la Bolsa de Santiago.
- [BCI](https://apimarket.bci.cl/): APIs públicas del BCI: cuentas, indicadores económicos e información bancaria.
- [queAFP](https://queafp.cl/api): Datos de las AFP chilenas.
- [queTalMiAfp](https://www.quetalmiafp.cl/AccederCuotas): Valores de cuotas de las AFP. Gratuita.
- [SII — Valores Económicos](https://zeus.sii.cl/admin/rss/sii_ind_rss.xml): RSS diario con valores de dólar, UF y UTM del SII.
- [Buda.com](https://api.buda.com/): API REST del exchange de criptomonedas Buda. Órdenes, mercado en tiempo real y más.

## 💳 Medios de Pago

- [BIN Lookup](https://teriko.cl/bin-lookup): Buscador de BINs chilenos y extranjeros con información actualizada.
- [Khipu](https://khipu.com/page/api-para-integradores): API REST para crear cobros y recibir pagos con Khipu.
- [Flow](https://www.flow.cl/apiFlow.php): Plataforma de pagos online con tarjetas de crédito y débito.
- [Kushki](https://docs.kushkipagos.com/api): Medios de pago globales con recepción en moneda local.
- [Reveniu](https://docs.reveniu.com/): API para cobros de suscripciones y pagos recurrentes.
- [Fintoc](https://docs.fintoc.com/reference/introduction): Iniciación de pagos por transferencias bancarias, tarjetas y conciliación de movimientos.

## 🤝 Cómo colaborar

Este listado se mantiene gracias a la comunidad, así que cualquier aporte es bienvenido. Algunas formas de ayudar:

- **¿Conoces una API que no está?** Abre un Pull Request agregándola en la categoría que corresponda (o crea una nueva sección si no encaja en ninguna). Mantén el mismo formato: `[Nombre](link): Descripción corta.`
- **¿Encontraste un link caído o una API que ya no funciona?** Abre un Issue avisando, o directamente muévela a la tabla de "APIs Deprecadas" con un PR.
- **¿La descripción de alguna API está mala o desactualizada?** También se agradecen las correcciones, por mínimas que sean.

Antes de mandar el PR, chequea que:

1. La API sea de origen chileno o esté fuertemente relacionada con datos/servicios de Chile.
2. El link funcione y lleve a documentación real (no a un sitio genérico).
3. No esté repetida en otra sección.

No hace falta que sea nada elaborado, con que el dato esté bien y el link funcione basta. Toda ayuda suma 🙌
